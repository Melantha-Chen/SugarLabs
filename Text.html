Text and Font Styling

  • Back
color Sets the color of the text.
font-family Specifies the typeface to be used for the text.
font-size Sets the size of the font.
font-weight Determines the weight (thickness) of characters in text. Common values are normal, bold, bolder, etc.
font-style Specifies the font style. Common values are normal, italic, and oblique.
text-align Aligns the text in an element. Common values are left, right, center, and justify.
text-decoration Adds decoration to text. Common values are none, underline, overline, and line-through.
text-transform Controls the capitalization of text. Common values are uppercase, lowercase, and capitalize.
line-height Specifies the amount of space above and below inline elements.
