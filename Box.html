Box Model

  • Back
margin Sets the space around elements.
padding Sets the space between the content of the element and its border.
border Sets the border around elements. It can have values defining its width, style, and color.
width Specifies the width of the element.
height Specifies the height of the element.
box-sizing Defines how the width and height are calculated, whether they include padding and borders or not. Common values are content-box and border-box.
