Flexbox and Grid

display: flex Indicates that the container is a flex container.
flex-direction Determines the direction of the flex items within the flex container.
justify-content Aligns flex items along the main axis.
align-items Aligns flex items along the cross axis.
display: grid Indicates that the container is a grid container.
grid-template-columns, grid-template-rows Defines the columns and rows of the grid.
grid-gap Defines the size of the gap between the rows and columns.
