Positioning and Display

  • Back
position Specifies the type of positioning for an element. Common values are static, relative, absolute, fixed, and sticky.
top, bottom, left, right Determines the position of the element based on its closest non-static positioned ancestor.
display Specifies how an element should be displayed. Common values are block, inline, inline-block, flex, grid, none, etc.
z-index Specifies the stack order of an element. Useful for controlling which element appears on top in case of overlap.
float Positions an element to the left or right, allowing other elements to wrap around it. Common values are left, right, and none.
clear Specifies what elements can float beside the cleared element. Common values are left, right, both, and none.
